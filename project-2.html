Realtime Stratergy Game (Prototype)

2025

I built a prototype real-time strategy game using Pygame. Players can select troops, move them, and attack in real-time. The game handles troop movement, combat interactions, and basic AI for enemy units. I focused on making the gameplay responsive and fast-paced, avoiding long delays or idle animations that slow down the action. This project was my first foray into game programming beyond static demos, and it taught me about game loops, event handling, and real-time logic in Python.
